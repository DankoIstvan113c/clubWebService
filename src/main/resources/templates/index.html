REST API webszolgáltatásai

Github repository

Metódus URL Szolgáltatás
GET /categories Kategóriák listája
GET /categories/{id}/clubs Egy kategória szórakozó helyei
POST categories Egy kategória létrehozása
DELETE categories/{id} Egy kategória törlése
PATCH categories/{id}/{agelimit} Egy kategória módosítása (alsó korhatár)
GET clubs Minden klub
GET clubs/{id} Egy szórakozóhely
POST clubs Új szórakozóhely felvitele
